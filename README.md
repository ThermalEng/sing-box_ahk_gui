# sing-box_ahk
用ahk做的一个sing-box的简单入口界面，只是提供一个gui入口，方便执行后台命令。
规则要预先手撸撸好。
退出是用标准的ctrl+c退出，没有暴力kill，防止路由表残留。
不能手动切换，只能退出重进，因为在win下ctrl+c无法只退出代理不退出界面。
注意要兼容性选项选择管理员模式。
